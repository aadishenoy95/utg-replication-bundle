# UTG

**Citable record:** [https://doi.org/10.5281/zenodo.19491557](https://doi.org/10.5281/zenodo.19491557)

UTG is a proposed unification framework for gravity, temporal structure, and quantum-sector behavior. This repository contains the current frozen UTG evidence package, including summaries, protocols, replication assets, and the code used to test the main laws.

## Current State

The current frozen UTG package contains four successful predictions:

1. **Two-Channel Source Law**
   - `rho = 8.736264e-01`
   - `p = 1.499925e-04`
2. **Phase-Morphology Law**
   - `rho = 6.648352e-01`
   - `p = 9.599520e-03`
3. **Phase-Strength Law**
   - `rho = -5.172932e-01`
   - `p = 1.049948e-02`
4. **Quantum Interference Law**
   - `rho = -8.956044e-01`
   - `p = 4.999750e-05`

These results were obtained under frozen protocols with fixed event sets, robustness checks, and replication-style reruns.

## Claim Boundary

What is justified from the current evidence:

- the strongest UTG law survives separate-environment replication, hostile robustness checks, and comparison against simpler baselines
- UTG currently has four frozen successful predictions in one theory framework
- gravity-side, temporal, and quantum-sector structure are all represented in the present evidence package

What is not yet fully established:

- replication by a different human analyst or team
- broader outside scrutiny from the physics community
- full outside-world acceptance of UTG
