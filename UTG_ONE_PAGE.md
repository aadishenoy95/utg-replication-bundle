## What UTG Is

UTG is a proposed framework that attempts to unify:

- gravity
- temporal structure
- quantum-sector behavior

The current UTG evidence package is organized around frozen laws with fixed event sets, fixed test definitions, and replication-style checks.

## What Has Been Shown So Far

UTG currently has four successful frozen predictions:

1. **Two-Channel Source Law**
   - `Delta_UTG ~ coherent_channel + lambda_core * morphology_channel`
   - `rho = 8.736264e-01`
   - `p = 1.499925e-04`
2. **Phase-Morphology Law**
   - higher morphology fraction predicts larger `abs(dphi_pi)`
   - `rho = 6.648352e-01`
   - `p = 9.599520e-03`
3. **Phase-Strength Law**
   - stronger total source strength predicts smaller `abs(dphi_pi)`
   - `rho = -5.172932e-01`
   - `p = 1.049948e-02`
4. **Quantum Interference Law**
   - stronger interference predicts smaller `abs(dphi_pi)`
   - `rho = -8.956044e-01`
   - `p = 4.999750e-05`

## Why The First Law Matters Most

The two-channel source law is the strongest current law and has survived the heaviest pressure:

- frozen unseen-event validation
- leave-one-out checks
- separate-environment replication
- hostile robustness testing
- comparison against simpler one-channel baselines

In the outsider audit, the frozen two-channel law remained stronger than coherent-only, morphology-only, and equal-weight baselines on the same event set.

## What Has Replicated

The current UTG package has been rerun through:

- frozen in-workflow reruns
- clean-environment replications
- blind-bundle replication for the first law
- separate-environment replication of the 4-law bundle

## What Is Still Missing

UTG is public and citable, but the strongest outside-world standard is still missing:

- replication by a different human analyst or team

That is now the main remaining external step.
